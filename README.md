# Real-Time Greeting System 🕐

Hey! This is one of my first Python projects.

It checks the current time and greets you accordingly:
- Good Morning ☀️ (before 12PM)
- Good Afternoon 🌤️ (12PM - 4PM)
- Good Evening 🌆 (4PM - 8PM)
- Good Night 🌙 (after 8PM)

## How to run it
Just run the Python file:
python Greetings-BasedOnTime.py

## What I learned
- Using the `time` module in Python
- Working with strftime() to get current hour
- Basic if/elif/else logic
